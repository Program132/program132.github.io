French Programming Language

FPL est un langage de programmation créer par Program.
L'objectif de projet est de créer un langage de programmation en français.
Cela signifi, que notre but est de proposer un langage de programmation avec des mots clés français!
Par exemple, généralement on utilisec le mot clé print pour envoyer un message dans la console.
Ici cela ne sera pas le cas, on utilisera envoyer ou send si on fait également en anglais!
Pourquoi ce projet? Depuis le lycée, Program remarque que le Python n'est pas forcément clair pour tout les étudiants.
C'est une raison pour la quelle il déçide de créer ce langage.

Service Modération Bot Discord

Program pensait à créer un service dans lequel il offre un bot discord de modération qui contient:
- ban
- kick
- warn
- mute
Ceci serait la première offre mais il pense à créer plusieurs offre avec de plus en plus de fonctionnalités dans le bot en question.
Ce service serait gratuit et payant (l'exemple ci dessus serait GRATUIT).
