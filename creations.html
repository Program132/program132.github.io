French Programming Language


FPL is a programming langage by Program.
The goal of this project: create French language.
It means, we will use french words in this programming language!
To give an example, mainly we use the word print to send a message in the console.
In this language, we will use envoyer (= send).
Why this project? Python can not be understandable for every students.

Froblox


Froblox is discord bot about Roblox Documentation but in French.
Since the 3rd April 2022, the bot is available : WebSite for Froblox !
Futhermore, the discord bot has not every pages of the documentation, only certain elements placed in priority by the Roblox French development community.
The bot has some services and a lot of basic informations like: the player's mouse, instance, and more!
